# Vision-Language Fashion Retrieval System

## Overview
This project is a multimodal fashion retrieval system built using BLIP and CLIP vision-language models.

The system allows users to upload clothing images and retrieve semantically similar fashion items using embedding similarity search.

## Features
- Clothing image upload
- BLIP caption generation
- CLIP embedding generation
- Cosine similarity retrieval
- Precomputed embeddings for scalability

## Models Used
- BLIP
- CLIP (openai/clip-vit-base-patch32)

## Dataset
Fashion Product Images Small Dataset (Kaggle)

## Technologies
- Python
- PyTorch
- HuggingFace Transformers
- Streamlit
